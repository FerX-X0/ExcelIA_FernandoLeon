# ExcelIA_FernandoLeon

# Proyecto Excel con IA

Autor: Fernando Leon

## Tema
Control de gastos personales de un estudiante.

## Pregunta
¿En qué categoría se gasta más dinero y cómo varían los gastos por mes?

## Herramientas usadas
- Excel
- Ideas en Excel (IA)
- Gráficos automáticos

## Resultados
Se identificaron las categorías con mayor gasto y se analizaron tendencias mensuales.
